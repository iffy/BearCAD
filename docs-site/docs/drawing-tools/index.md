# Drawing Tools

Opening a [drawing](/docs/tools/drawing) switches to the **technical drawings** workbench.
Its toolbar acts on the page instead of the model — Select, Dimension, and Text share a
name with a modeling tool but do a different job here.


| | Tool | Shortcut | What it does on a drawing |
|---|---|---|---|
| <img src={useBaseUrl("/img/icons/select.svg")} width="22" /> | [Select](/docs/drawing-tools/select) | — | Pick and drag views, text notes, and dimensions around the page. |
| **＋** | [Add view](/docs/drawing-tools/add-view) | — | Drop a projection of a body or sketch onto the page. |
| | [Aligned view](/docs/drawing-tools/aligned-view) | — | Add a projection lined up with an existing one. |
| <img src={useBaseUrl("/img/icons/dimension.svg")} width="22" /> | [Dimension](/docs/drawing-tools/dimension) | `D` | Show or hide a view's length, diameter, and angle dimensions. |
| <img src={useBaseUrl("/img/icons/text.svg")} width="22" /> | [Text](/docs/drawing-tools/text) | `T` | Add page **text notes** (with live parameter variables). |

The [Drawings](/docs/tools/drawing) page covers the sheet itself: creating one, page
size, exporting, and scripting.
